NovaEdit Troubleshooting Guide

Overview

This guide helps solve common NovaEdit problems.

⸻

NovaEdit Does Not Start

Possible solutions:

* Restart your device
* Reinstall NovaEdit
* Check system requirements
* Review crash logs

⸻

Application Crashes

Try:

1. Restart NovaEdit.
2. Check for updates.
3. Disable recently added plugins.
4. Review crash reports.

⸻

Slow Performance

Possible solutions:

* Close unused projects
* Disable unnecessary plugins
* Reduce AI usage
* Update graphics drivers

⸻

Missing AI Features

Check:

* Nova AI is installed.
* AI is enabled.
* Required dependencies exist.
* API settings are correct.

⸻

Plugin Problems

If a plugin causes issues:

1. Disable the plugin.
2. Restart NovaEdit.
3. Remove the plugin if necessary.

⸻

Update Problems

Try:

* Checking internet connection.
* Running NovaEdit with required permissions.
* Manually installing the latest version.

⸻

Reporting Issues

If problems continue:

Provide:

* NovaEdit version
* Operating system
* Error messages
* Steps to reproduce
