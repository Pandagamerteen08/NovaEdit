NovaEdit User Guide

Overview

This guide explains common NovaEdit features and workflows.

⸻

Projects

Projects allow users to organize related files together.

A project may contain:

* Source code
* Assets
* Documentation
* Configuration files

Projects help keep development organized.

⸻

File Management

NovaEdit allows users to:

* Open files
* Create files
* Rename files
* Organize folders
* Manage project structures

⸻

Editing Files

NovaEdit provides tools for:

* Text editing
* Syntax highlighting
* Searching
* Replacing text
* Multiple open files

⸻

Tabs

Tabs allow users to work with multiple files at the same time.

Users can:

* Switch between files
* Close files
* Reopen recently used files

⸻

Search

Search tools allow users to quickly find:

* Words
* Code sections
* Files
* Project content

⸻

Themes

NovaEdit supports customization through themes.

Themes may change:

* Colors
* Editor appearance
* Interface style

See:

THEMES.md

⸻

Languages

NovaEdit supports programming language features.

Initial supported languages:

* Python
* JavaScript
* HTML
* CSS
* JSON
* Markdown

More languages may be added later.

⸻

AI Assistance

Nova AI provides optional assistance features.

Possible features:

* Explaining code
* Suggesting improvements
* Helping understand projects

AI features can be controlled through settings.

⸻

Privacy

NovaEdit is designed with privacy in mind.

The application does not require:

* Accounts
* Cloud storage
* Online connection

unless users choose optional online features.
