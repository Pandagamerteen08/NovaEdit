# NovaEdit Update System

## Overview

NovaEdit includes a built-in update system designed to keep users on the latest stable release.

---

# Update Features

The updater can:

- Check for updates
- Compare versions
- Download updates
- Install updates

---

# Update Checking

Users can configure update checking.

Possible options:

- Every startup
- Every 5 minutes
- Every hour
- Daily
- Manually
