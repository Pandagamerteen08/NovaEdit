NovaEdit Development Guide

Overview

This guide explains development information for contributors.

⸻

Development Goals

NovaEdit development focuses on:

* Clean architecture
* Maintainable code
* Privacy
* Performance
* Cross-platform support

⸻

Development Environment

Recommended tools:

* Python
* Git
* Code editor
* Virtual environment

⸻

Project Structure

Main source:

src/novaedit/

Major systems:

* Editor
* Project manager
* Settings
* AI
* Language support

⸻

Python Guidelines

Follow:

* PEP 8
* Clear naming
* Type hints where useful
* Documentation for important systems

⸻

Testing

Developers should test:

* New features
* Existing functionality
* Platform compatibility

⸻

Pull Requests

Before submitting:

* Test changes
* Update documentation
* Explain changes clearly
* Follow contribution rules

⸻

Future Development Areas

Possible areas:

* Plugin system
* Theme system
* AI improvements
* Language integrations
* Performance optimization
