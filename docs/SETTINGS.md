NovaEdit Settings Guide

Overview

NovaEdit settings allow users to customize their experience.

Settings are stored locally unless optional features require otherwise.

⸻

General Settings

Possible options:

* Startup behavior
* Default folders
* Interface preferences
* Language preferences

⸻

Editor Settings

Editor settings may include:

* Font size
* Font family
* Line numbers
* Word wrapping
* Tab spacing
* Auto-save

⸻

Appearance Settings

Appearance settings may include:

* Themes
* UI scaling
* Editor colors
* Icon styles

⸻

AI Settings

AI settings may include:

* Enable or disable Nova AI
* Select AI provider
* Manage API keys
* Conversation storage
* Project memory settings

⸻

Update Settings

Users may configure:

* Automatic update checking
* Update frequency
* Notifications

Possible update intervals:

* Every startup
* Every few minutes
* Daily
* Manually

⸻

Privacy Settings

Users may control:

* Telemetry
* Crash reporting
* Optional analytics

Privacy-related features are designed to be transparent.

⸻

Plugin Settings

Future plugin settings may include:

* Installed plugins
* Plugin permissions
* Enable or disable plugins

⸻

Resetting Settings

Users may restore default settings through the settings menu.

Resetting settings does not automatically delete projects.
