Nova AI Guide

Overview

Nova AI is NovaEdit’s optional artificial intelligence assistant.

Nova AI is designed to help users understand code, improve productivity, and learn programming concepts while keeping user privacy as a priority.

Nova AI is optional and can be disabled completely.

⸻

AI Philosophy

Nova AI follows these principles:

* User control first
* Privacy-focused design
* Transparent behavior
* No hidden data collection
* Local-first functionality

⸻

Installation

Nova AI may be installed:

* During NovaEdit installation
* Through NovaEdit settings
* Through optional components

Nova AI is not required for NovaEdit to function.

⸻

AI Modes

Nova AI may support multiple modes.

Local AI

Local AI runs directly on the user’s device.

Benefits:

* Increased privacy
* No required account
* Offline availability
* User-controlled data

Requirements may include:

* Additional storage
* More system resources

⸻

External API Mode

Users may optionally connect their own API keys.

Supported providers may vary.

When using external AI services:

* The provider’s privacy policy applies.
* Users control their own API keys.
* Users should review provider terms.

⸻

Project-Based AI Memory

Nova AI may organize conversations by project.

Examples:

* Game project assistance
* Website project assistance
* Application development assistance

Project memory can help Nova AI understand context.

⸻

Conversation Storage

Users control AI conversation storage.

Possible options:

* Save conversations
* Automatically delete conversations
* Manually delete conversations

⸻

AI Limitations

AI responses may contain:

* Incorrect information
* Outdated information
* Programming mistakes
* Security problems

Always review AI-generated suggestions before using them.

⸻

AI Privacy

Nova AI does not automatically upload:

* Projects
* Source code
* Documents

unless the user chooses an external AI service or feature requiring it.

⸻

Future Features

Possible future features:

* Code explanations
* Documentation generation
* Debugging assistance
* Project analysis
* Learning assistance
* Refactoring suggestions
