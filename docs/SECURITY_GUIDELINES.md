# NovaEdit Security Guidelines

## Overview

These guidelines apply to contributors creating:

- Plugins
- Themes
- Extensions
- Integrations

---

# Plugin Security

Plugins should:

- Respect user privacy
- Avoid unnecessary permissions
- Not collect data without permission
- Not contain malicious code

---

# Malware Prevention

Official plugins may be checked using:

- Malware scanners
- VirusTotal
- Manual review

---

# User Safety

Users should:

- Download from trusted sources
- Review permissions
- Avoid unknown plugins

---

Security problems should be reported privately.
