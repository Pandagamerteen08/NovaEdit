# NovaEdit Installer Details

## Overview

NovaEdit uses a custom cross-platform installation system designed to provide a simple installation experience across:

- Windows
- Linux
- macOS

The installer is responsible for installing NovaEdit, optional components, and required files.

---

# Installer Names

The official installer name is currently planned as: 

Windows: NovaEdit_Setup.exe or NovaEdit_Installer.exe

macOS: NovaEdit_Setup.dmg or NovaEdit_Installer.pkg

Linux: NovaEdit_Setup.d or NovaEdit_Installer.exe
