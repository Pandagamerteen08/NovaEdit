NovaEdit Language Support

Overview

NovaEdit provides programming language support through editing tools and language integrations.

⸻

Initial Supported Languages

NovaEdit initially supports:

Python

Features:

* Syntax highlighting
* File support
* Code editing

⸻

JavaScript

Features:

* Syntax highlighting
* File support
* Web development workflows

⸻

HTML

Features:

* Syntax highlighting
* Web document editing

⸻

CSS

Features:

* Syntax highlighting
* Styling file support

⸻

JSON

Features:

* Formatting
* Syntax highlighting
* Configuration editing

⸻

Markdown

Features:

* Documentation editing
* Formatting support

⸻

Future Languages

Possible future additions:

* Java
* C#

Advanced features such as:

* Compiling
* Debugging
* Build tools

may require additional setup.

⸻

External Tools

Some languages require external software.

NovaEdit may detect missing requirements and provide recommendations.

⸻

Language Support Philosophy

NovaEdit focuses on providing excellent editing support first.

Advanced development features may be added over time.
