NovaEdit Changelog

All notable changes to NovaEdit will be documented in this file.

NovaEdit follows Semantic Versioning:

MAJOR.MINOR.PATCH

Version Format

Major Version

Major releases include significant changes, redesigns, major feature additions, or changes that may affect compatibility.

Example:

1.0.0

Minor Version

Minor releases add new features and improvements while maintaining compatibility.

Example:

0.2.0

Patch Version

Patch releases contain bug fixes, performance improvements, and smaller changes.

Example:

0.1.1
